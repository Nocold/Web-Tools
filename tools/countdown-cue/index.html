⏱️ 倒數提示音

在音樂的指定時間點疊上英文倒數語音(five… one),讓倒數結尾正好對齊該時間點; 還可在倒數前加上區段提示(Part A… 5… 4… 3… 2… 1)。全部在瀏覽器內合成,音檔不會上傳。

🎼 點擊選擇音樂 或拖曳到此處
支援 mp3 / wav / m4a 等常見音訊
倒數結尾會落在這些時間點。格式:秒、mm:ss 或 hh:mm:ss。設 0 代表在開頭前面加一段倒數。
1–10
%
試聽會直接播放「(第一個時間點的區段)… 5 … 4 … 3 … 2 … 1」,不需先選音樂。

          輸出成 MP3 時首次需下載 FFmpeg 核心(約 30MB,之後由瀏覽器快取)。
          若時間點小於倒數秒數(例如 0:00),會自動在開頭補上足夠的前導時間。
