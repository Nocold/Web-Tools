🎬 演出提詞機(v2)

類似歌詞提詞器的演出流程小工具:先把流程腳本(時間點 + 段落內容)編輯好, 播放時會用大字顯示目前段落、提前顯示下一段「預告」,並可選擇在段落開始前播放倒數提示音。 腳本可存成本機檔案,也可選配登入 Google 雲端硬碟存取。

☁️ Google 雲端硬碟(選用)
本站已內建 Google 登入設定,可直接按下方「登入 Google」授權,腳本會存到你自己的 Google 雲端硬碟(僅能存取本工具建立的檔案)。
自訂 Client ID
Google 規定 OAuth 用戶端必須綁定「已授權的網域」才能登入,無法做到完全免設定的萬用登入;若你是把這個工具另外架設在自己的網域(而不是本站內建的網域),需要自行到 Google Cloud Console 建立一組屬於你網域的 Client ID: 1. 建立專案 → 「API 和服務」→「憑證」→ 建立「OAuth 用戶端 ID」,類型選「網頁應用程式」。 2. 在「已授權的 JavaScript 來源」加入這個網站的網域(例如 https://your-user.github.io),不需要設定重新導向 URI。 3. 到「OAuth 同意畫面」新增範圍 https://www.googleapis.com/auth/drive.file(僅能存取本工具自己建立的檔案,不會讀取你雲端硬碟其他檔案)。 4. 把產生的 Client ID 貼到下方欄位並儲存,再按「登入 Google」。
也支援只有單一時間點的格式,例如「1:30 冷+檸」
